## 新版 CLI 快速入口

| 命令 | 说明 |
|------|------|
| `biliup server --auth` | 启动 WebUI 录制服务（最常用） |
| `biliup login` | 登录 B站 |
| `biliup upload` | 上传视频 |
| `biliup download` | 下载直播/视频 |
| `biliup list` | 列出已上传视频 |
| `biliup append` | 追加视频到已有稿件 |
| `biliup comments` | 下载视频评论 |
| `biliup reply` | 下载评论回复 |
| `biliup renew` | 刷新登录信息 |
| `biliup show` | 查看视频详情 |
| `biliup --version` | 查看版本号 |
| `biliup --help` | 查看完整帮助 |

详细参数说明请参阅 [CLI 命令行参考](/guide/configs/config.html)。
